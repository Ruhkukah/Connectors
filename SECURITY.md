# Security Policy

This repository is public. Operational trading data and sensitive configuration must not be published here.

Never commit or post in issues, pull requests, discussions, workflow logs, or attached artifacts:

- credentials or authentication material
- broker configuration files
- production logs
- certification logs
- broker latency measurements
- broker topology or routing data

If you need to report a problem that involves any of the above:

- sanitize the reproduction first
- reduce it to non-sensitive placeholders or synthetic data
- keep public issues free of operational details

This repository does not use the public issue tracker as a place to disclose secrets or live operational data.
